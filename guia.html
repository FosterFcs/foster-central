Bem-vindo ao Foster Obras

O Foster Obras é uma plataforma digital completa para gerenciamento de obras e geração de Relatórios Diários de Obra (RDO). Desenvolvida especialmente para engenheiros e arquitetos, oferece uma interface intuitiva e funcionalidades robustas para acompanhar projetos em tempo real.

📊 Dashboard Inteligente

Visualize estatísticas em tempo real: total de obras, obras ativas e RDOs registrados.

🏗️ Gestão de Obras

Cadastre e gerencie múltiplas obras com detalhes completos: cliente, endereço, orçamento e cronograma.

📋 Relatórios Diários

Crie RDOs detalhados registrando clima, equipe, atividades, materiais e ocorrências.

📈 Acompanhamento

Monitore o avanço físico de cada obra com barras de progresso e relatórios consolidados.

📱 PWA Responsivo

Acesse de qualquer dispositivo. Funciona como um app instalável no seu celular.

🔐 Autenticação Segura

Login seguro com e-mail e senha. Dados protegidos no Supabase.

🔐 Sistema de Autenticação

O Foster Obras oferece um sistema robusto de autenticação com duas opções: Login e Cadastro.

Modo Demo: A aplicação inicia com dados de demonstração já carregados, permitindo explorar todas as funcionalidades sem necessidade de login.

Tela de Login

Campos de Login:

  • E-mail profissional: Seu e-mail corporativo para acesso
  • Senha: Mínimo de 8 caracteres para segurança
  • Botão "ENTRAR": Realiza a autenticação no sistema
  • Link "Criar conta": Redireciona para o formulário de cadastro

Tela de Cadastro

Campos de Cadastro:

  • Nome completo: Seu nome profissional
  • CREA/CAU: Número do registro profissional (opcional)
  • E-mail: E-mail corporativo para cadastro
  • Senha: Mínimo de 8 caracteres
  • Confirmação: Após cadastro, 7 dias grátis são ativados

🏗️ Gestão de Obras

A seção de Obras permite cadastrar, visualizar e gerenciar todos os seus projetos de construção.

Como Cadastrar uma Nova Obra

  • Clique no botão "+ NOVA OBRA" na aba de Obras
  • Preencha os dados obrigatórios: Nome, Cliente e Endereço
  • Adicione informações complementares: ART/RRT, Área, Orçamento, etc.
  • Selecione o status inicial (Em Andamento, Pausada, Planejamento)
  • Clique em "SALVAR" para registrar a obra

📋 Relatório Diário de Obra (RDO)

Os RDOs são registros diários essenciais que documentam todas as atividades, condições climáticas e ocorrências de cada obra.

Como Criar um RDO

  • Acesse a aba "RDO" no menu de navegação
  • Selecione a obra para a qual deseja registrar o RDO
  • Clique em "+ NOVO RDO"
  • Preencha os dados: data, clima, equipe, atividades, etc.
  • Clique em "SALVAR RDO" para registrar

📄 Relatório Consolidado

A seção de Relatório permite gerar documentos consolidados com todas as informações de uma obra em um período específico.

Filtros de Relatório

Exemplo de Relatório

RELATÓRIO DE OBRA
Obra Residência Família Santos
Cliente Carlos Santos
Período 01/02/2026 a 28/02/2026
Avanço 42%
RDOs do Período:
• 01/02 - Fundação iniciada (8 pedreiros)
• 02/02 - Estrutura em andamento (10 operários)
• 03/02 - Alvenaria iniciada (6 pedreiros)
Funcionalidades do Relatório: Visualização de todos os RDOs registrados no período, consolidação de dados, estatísticas de avanço e exportação em PDF.
